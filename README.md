# JavaScript: Практический курс

Курс из 32 модулей для изучения JavaScript с нуля. Каждый модуль -- самостоятельное задание с теорией, тестами и подсказками. Вы пишете код, запускаете тесты -- и сразу видите, правильно ли решение.

## Быстрый старт

1. Проверьте [Node.js](https://nodejs.org/) (версия 18 или выше)

```bash
node -v
```
2. Установите зависимости:

```bash
npm install
```

3. Запустите тесты в браузере:

```bash
npm run test
```

4. Откройте первый модуль: `module-01/README.md`

---

## Как устроен курс

Каждый модуль -- отдельная папка с файлами:

| Файл | Назначение |
|------|------------|
| `README.md` | Теория и описание задания |
| `index.js` | Ваш файл -- пишите решение здесь |
| `index.spec.js` | Тесты (не изменяйте) |

Порядок работы: прочитайте теорию в README, напишите код в `index.js`, запустите тесты. Все тесты зелёные -- модуль пройден.

## Как работать с заданием

1. Откройте `module-XX/README.md` и прочитайте теорию
2. Откройте `module-XX/index.js` -- там заготовка с комментариями
3. Напишите своё решение
4. Запустите тесты:

```bash
# Один модуль
npx vitest module-01/index.spec.js

# Один модуль в браузере
npx vitest module-01/index.spec.js --ui

# Все модули
npm run test
```

5. Если застряли -- в README каждого модуля есть 3 уровня подсказок (от общей идеи до почти-решения)

## Тестирование с Vitest

Проект использует [Vitest](https://vitest.dev/) для запуска тестов.

### Команды

| Команда | Описание |
|---------|----------|
| `npm run test` | Запуск тестов в браузере (рекомендуется) |

### Vitest UI

Запустив `npm run test`, вы получите веб-интерфейс с тремя вкладками:

- **Report** -- сводка по тестам: сколько прошло, сколько упало
- **Code** -- исходный код тестов и связь с проверяемыми функциями
- **Console** -- логи и сообщения, полезно для отладки через `console.log`
