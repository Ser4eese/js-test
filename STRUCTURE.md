  Блок 1: Основы JavaScript (5 модулей, ~3 занятия)

  Цель: Студенты понимают базовый синтаксис JS

  1. module-01: Переменные и типы данных (let, const, примитивы)
  2. module-02: Условия и операторы сравнения
  3. module-03: Массивы — основы работы (создание, индексы, length)
  4. module-04: Циклы (for, while)
  5. module-05: Функции (declaration, expression, arrow)

  Блок 2: Объекты и инструменты (5 модулей, ~3 занятия)

  Цель: Работа с объектами, методами массивов + знакомство с Git и npm

  6. module-06: Методы массивов (push, pop, shift, unshift)
  7. module-07: Объекты (создание, доступ к свойствам)
  8. module-08: Методы объектов и spread оператор
  9. module-09-git: Git основы (теоретический модуль с практикой)
  10. module-10-npm: npm, package.json, установка зависимостей

  Блок 3: HTML и DOM (4 модуля, ~3 занятия)

  Цель: Создание интерфейсов и работа с ними

  11. module-11: HTML основы + создание формы
  12. module-12: DOM - что это, querySelector, изменение элементов
  13. module-13: События (addEventListener, event object)
  14. module-14: Сбор данных из формы и валидация

  Блок 4: Асинхронность (5 модулей, ~3 занятия)

  Цель: Понимание асинхронности и Event Loop

  15. module-15: Event Loop (теория + setTimeout)
  16. module-16: Callbacks
  17. module-17: Promises (создание, then/catch)
  18. module-18: async/await
  19. module-19-debug: Debugging в браузере и Node.js

  Блок 5: HTTP и Backend (5 модулей, ~4 занятия)

  Цель: Создание backend на чистом Node.js

  20. module-20: Что такое HTTP (request/response, методы, статусы)

  22. module-22: Создание простого сервера (http модуль)
  23. module-23: Routing и обработка разных путей
  24. module-24: Query параметры и POST запросы

  Блок 6: Fullstack приложение (3 модуля, ~3 занятия)

  Цель: Связать фронт и бэк

  25. module-25: CORS и связка фронт-бэк
  26. module-26: Полноценное TODO приложение (фронт + бэк)
  27. module-27: Работа с файлами или простой БД (JSON файл)

  Блок 7: Frameworks (4 модуля, ~2 занятия)

  Цель: Знакомство с современными инструментами

  28. module-28: Введение в Express (переписываем сервер)
  29. module-29: Введение в React (компоненты, JSX, useState)
  30. module-30: React + fetch (получение данных)
  31. module-31: Финальный проект

  3. Практика: От простых функций к полноценному приложению
  4. Текстовые описания: В каждом README будет:
    - Теория (зачем это нужно)
    - Примеры кода

    - Подсказки для разных уровней студентов
